# Баг-репорты

В этой папке представлены примеры баг-репортов, созданных в учебных целях и для портфолио.

## Содержание

- Ошибки пользовательского интерфейса (UI Bugs)
- Функциональные ошибки (Functional Bugs)
- Ошибки, найденные во время регрессионного тестирования (Regression Bugs)
- Ошибки мобильной веб-версии (Mobile Web Bugs)

Все примеры анонимизированы и не содержат конфиденциальной информации.
