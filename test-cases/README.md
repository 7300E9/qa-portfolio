# Test Cases

This folder contains examples of manual test cases created for learning and portfolio purposes.

## Contents

- Authentication
- Registration
- User Profile
- Product Catalog
- Search
- Shopping Cart
- Checkout
- Order History

## Purpose

The test cases demonstrate my ability to write clear test documentation, describe testing scenarios, define expected results, and verify application functionality.

> All examples are created for educational and portfolio purposes and do not contain confidential information.


# Тест-кейсы

В этой папке представлены примеры ручных тест-кейсов, созданных для обучения и портфолио.

## Содержание

- Авторизация
- Регистрация
- Профиль пользователя
- Каталог товаров
- Поиск
- Корзина
- Оформление заказа
- История заказов

## Назначение

Тест-кейсы демонстрируют мое умение писать понятную тестовую документацию, описывать сценарии тестирования, формулировать ожидаемые результаты и проверять функциональность приложения.

Все примеры созданы исключительно для обучения и портфолио и не содержат конфиденциальной информации.
